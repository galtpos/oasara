# How to Get Freedom Dollars (fUSD) - Tutorial Script

## Advisory Board Design Principles Applied

- **Joseph Coughlin**: Large text, clear steps for 55+ users
- **Don Norman**: One action per screen, clear affordances
- **Ruth Clark**: Segment into chunks, visual + narration
- **Jony Ive**: Minimal - only essential information
- **Paulo Freire**: Emphasize liberation and self-sovereignty
- **Salman Khan**: Mastery-based progression
